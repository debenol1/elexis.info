Elexis

Elexis is an Eclipse RCP program for all aspects of a medical practice: electronic medical record (EMR), laboratory findings etc., as well as accounting, billing (swiss TARMED-System, other systems to be developped) and other daily work.

Error in the website?

You can send an error/suggestion to the maintainer of this website Niklaus Giger .

It is also possible to view the source code of this site and test major customizations locally. For this you need installed versions of ruby and git on your computer.

After that you should make a running copy on the command line with the following commands.

gem install jekyll-polyglot jekyll-redirect-from
git clone https://github.com/elexis/elexis.github.com.git
cd elexis.github.com
jekyll serve

If all goes well, you can see a local copy of this page at http://127.0.0.1:4000.

Niklaus often uses https://www.deepl.com for translations.
